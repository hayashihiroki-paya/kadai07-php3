# 制作アプリのタイトル：ラノベならべ さくらサーバー利用

## 制作アプリの説明
- ライトノベルをタイトルで検索して探せます
- その中から自分の好きな作品をお気に入りに保存できます
- 保存した作品の、特に好きなところを保存することができます
- ユーザーごとの傾向、本ごとの傾向を比較し、マッチの評価値を計算しておすすめに表示できるようになりました

## 工夫した点・こだわった点
- マッチ評価値の計算を行い、上位から表示させる機能はマストで実装したかったところなので間に合ってよかったです
