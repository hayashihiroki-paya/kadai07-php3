# 制作アプリのタイトル：ラノベならべ DB活用版

## 制作アプリの説明
- ライトノベルをタイトルで検索して探せます
- その中から自分の好きな作品をお気に入りに保存できます
- 保存した作品の、特に好きなところを保存することができます
- 作品ごとにどういったところが好まれているのかを統計データで表示できるようにしました
- <img width="3240" height="2160" alt="Image" src="https://github.com/user-attachments/assets/9f84591d-e691-4310-a7de-23ecef8deb6e" />

## 工夫した点・こだわった点
- 前回の課題からDB保存に変更しました
- 本ごとに全ユーザーからの入力情報を統計して出すことができました

## 次回トライしたこと（または機能）
- 統計データをもっとうまく活用しようと思ってましたがまとまらずに、表示方法も前回からほとんど変わっていないのでアイデアをまとめていきたいです
- 作品ごとに全ユーザーからの入力情報を取得できるようになったので、ユーザーの傾向と作品の傾向をマッチングする流れをテストケースだけでもいいので実装してみようと思います
